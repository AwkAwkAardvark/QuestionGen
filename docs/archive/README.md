# Docs Archive

This directory holds historical planning and audit documents that no longer define the live repo contract.

Use archived docs for rationale, migration history, or old review context only. Current product policy, workflow policy, and stable interface commitments belong in `IMPLEMENTATION_PLAN.md` and the active contract docs under `docs/`.

Tracked archive docs live directly under `docs/archive/`. Local-only agent history belongs under the ignored `docs/archive/local/` area so branch-by-branch working notes do not get pushed to the remote repository.
